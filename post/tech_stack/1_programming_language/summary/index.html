现代C++

作为一门编程工具的最核心，C++的语言和标准库一直在扩展和强化。从C++11开始，C++14、C++17、C++20、C++23以及未来的版本，业界习惯称之为“现代C++(Modern C++)”，用以区别“传统C++”。现代C++远不止是修修补补，其所考虑的新功能，使这门经典语言脱胎换骨。
